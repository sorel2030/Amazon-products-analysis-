# Amazon Products and Customer Reviews Analysis

## Project Overview
This project explores Amazon product data and customer reviews using Python for exploratory data analysis and data visualization.

The objective was to identify trends in product ratings, customer engagement, and pricing patterns to better understand customer behavior and product performance.

---

## Dataset
The dataset contains information about Amazon products, including:
- Product categories
- Ratings
- Review counts
- Prices
- Customer reviews

---

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Project Steps
1. Data cleaning and preprocessing
2. Exploratory data analysis (EDA)
3. Visualization of customer ratings and review trends
4. Analysis of pricing and product popularity

---

## Key Insights
- Products with higher ratings generally received more customer engagement.
- Certain product categories showed significantly higher review activity.
- Data visualization helped identify trends and potential business insights.
